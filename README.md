# Stranded Shores
## How to build
1. Clone the repo
2. Download [Godot](https://godotengine.org/download) 4.5 or later.
3. Open Godot, click on Import and select the `project.godot` file inside the repository folder.
   
   ![Import](https://github.com/KristianNikolov07/diploma-project/blob/main/READMEImages/BuildInstructions/Import.png)
   ![ProjectFile](https://github.com/KristianNikolov07/diploma-project/blob/main/READMEImages/BuildInstructions/ProjectFile.png)
4. Once loaded click on `Project -> Export`
   
   ![ExportButton](https://github.com/KristianNikolov07/diploma-project/blob/main/READMEImages/BuildInstructions/ExportButton.png)  
5. Select your platform and click on `Export Project`
   
   ![Export](https://github.com/KristianNikolov07/diploma-project/blob/main/READMEImages/BuildInstructions/Export.png)


## How to mod
1. Download [Godot](https://godotengine.org/download) 4.5 or later.
2. Clone the repo
3. Open Godot, click on Import and select the `project.godot` file inside the repository folder.
   
   ![Import](https://github.com/KristianNikolov07/diploma-project/blob/main/READMEImages/BuildInstructions/Import.png)
   ![ProjectFile](https://github.com/KristianNikolov07/diploma-project/blob/main/READMEImages/BuildInstructions/ProjectFile.png)
4. Make the changes you want to make
5. When done click on `Project -> Export`
   
   ![ExportButton](https://github.com/KristianNikolov07/diploma-project/blob/main/READMEImages/BuildInstructions/ExportButton.png)  
6. Select the preset named `mod`

   ![ModPreset](https://github.com/KristianNikolov07/diploma-project/blob/main/READMEImages/ModInstructions/ModPreset.png)
7. Go to the `Resources` tab

   ![ResourcesSection](https://github.com/KristianNikolov07/diploma-project/blob/main/READMEImages/ModInstructions/ResourcesSection.png)
8. Make sure the all the files you've modified are selected
9. Click on `Export PCK/ZIP` and make sure to save the file with the `.pck` extention.

    ![ExportPCKButton](https://github.com/KristianNikolov07/diploma-project/blob/main/READMEImages/ModInstructions/ExportPCKButton.png)
