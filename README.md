# How to build
1. Clone the repo
2. Download [Godot](https://godotengine.org/download) 4.5 or later.
3. Open Godot, click on Import and select the `project.godot` file inside the repository folder.
   
   ![Import](https://github.com/KristianNikolov07/diploma-project/blob/main/READMEImages/BuildInstructions/Import.png)
   ![ProjectFile](https://github.com/KristianNikolov07/diploma-project/blob/main/READMEImages/BuildInstructions/ProjectFile.png)
4. Once loaded click on `Project -> Export`
   
   ![ExportButton](https://github.com/KristianNikolov07/diploma-project/blob/main/READMEImages/BuildInstructions/ExportButton.png)  
5. Select your platform and click on `Export Project`
   
   ![Export](https://github.com/KristianNikolov07/diploma-project/blob/main/READMEImages/BuildInstructions/Export.png)
